# Conversions #

|       | Dry     | Liquid |
|:------|:--------|:-------|
| 1   C | 16  T   | 8   oz |
| 1/2 C | 8   T   | 4   oz |
| 1/3 C | 5T & 1t | 2.7 oz |
| 1/4 C | 4   T   | 2   oz |
| 1/8 C | 2   T   | 1   oz |
| 1   T | 3   t   | 1/2 oz |


# Substitutions #

**Olive Oil**			use in place of butter

**Whole Wheat Flour**	use in place of flour, adding 1 T wheat gluten for texture
